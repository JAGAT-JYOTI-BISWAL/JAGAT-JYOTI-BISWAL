### 3. Context-Aware Intelligence System — RAG-Based AI Backend

- **Repo:** https://github.com/JAGAT-JYOTI-BISWAL/context-aware-intelligence-system
- **Live Demo:** https://context-aware-intelligence-system.onrender.com  
⚠️ Free-tier deployment — may take 30–60 seconds to wake on first load.

**Highlights**

- Implemented semantic retrieval using HuggingFace embeddings and cosine similarity
- Designed prompt construction with ranked retrieved context injected into LLM input
- Built single-service backend flow for embedding generation, retrieval, prompt construction, and response delivery
- Demonstrated how retrieval changes answer quality in a visible, system-level way
